# Python

Scripts simples em Python voltados para prática de lógica, entrada de dados e menus no terminal.

## Destaque

O arquivo mais representativo desta pasta hoje é `calculator.py`, porque já entrega um fluxo completo de interação em terminal com múltiplas operações.

## O que é vitrine e o que é estudo

- `calculator.py` deve ser lido como a principal peça desta pasta
- `cardapio.py` entra mais como rascunho de estudo
- `trabalhos_de_faculdade e basico de python.py` funciona como coleção de exercícios básicos

## Arquivos

### `calculator.py`
- calculadora interativa em terminal
- operações de soma, subtração, multiplicação e divisão
- inclui validação simples para divisão por zero

### `cardapio.py`
- experimento inicial com entrada de pedidos em terminal
- ainda é um script incompleto
- serve mais como rascunho de estudo do que como peça forte de portfólio

### `trabalhos_de_faculdade e basico de python.py`
- conjunto de exercícios básicos
- soma de variáveis
- cálculo de média
- entrada de nome e idade
- condicional de aprovação, recuperação e reprovação

## Leitura da pasta

Esta pasta mostra prática de fundamentos e constância de estudo. O foco aqui não é complexidade de produto, e sim construção de base com Python.

## Como executar

```bash
python3 calculator.py
```
