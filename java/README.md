# Java

Exercícios e testes iniciais em Java, com foco em sintaxe, estrutura de classes e entrada de dados.

## Destaque

O principal item desta pasta hoje é o projeto `atividades_praticas_1/`, porque ele organiza os exercícios em uma estrutura mais próxima de projeto real com Maven.

## O que é vitrine e o que é estudo

- `atividades_praticas_1/` é o item mais relevante desta pasta hoje
- `Teste.java` é só um teste mínimo de compilação
- a leitura correta aqui é de formação inicial em Java, não de projeto pronto

## Estrutura

### `Teste.java`
- arquivo mínimo para testar compilação e saída no terminal

### `atividades_praticas_1/`
- projeto Maven com exercícios introdutórios
- reúne classes independentes para questões de prática

Veja mais em [atividades_praticas_1/README.md](atividades_praticas_1/README.md).

## Como executar

Arquivo simples:

```bash
javac Teste.java
java Teste
```

Projeto Maven:

```bash
cd atividades_praticas_1
mvn package
```

## Leitura da pasta

Esta pasta ainda representa um estágio inicial em Java. O ponto forte aqui é a organização do estudo e a prática com a estrutura da linguagem, mais do que a complexidade dos exercícios em si.
