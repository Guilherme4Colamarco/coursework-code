# atividades_praticas_1

Projeto Maven com exercícios introdutórios de Java.

## Papel dentro do portfólio

Este projeto entra mais como estudo estruturado do que como vitrine principal.

Ele ajuda a mostrar:

- contato inicial com Maven
- organização de arquivos Java em projeto
- prática de exercícios curtos por classe

## Leitura honesta

Hoje este projeto deve ser lido como coleção de exercícios em andamento. Ele ajuda a mostrar organização, prática com Java e familiaridade inicial com Maven, mas ainda não funciona como vitrine de projeto finalizado.

## Objetivo

Servir como coleção de exercícios curtos, um arquivo por questão, para praticar:

- entrada com `Scanner`
- operações matemáticas
- condicionais
- organização básica de projeto Java
